# Aillyn: Flexible generator children's storybook

**Aillyn** is a flexible children's storybook generator that brings imagination to life. Kids can create their own magical adventures by adding custom characters exactly as they want them!

## Features

* **Unlimited Custom Characters:** Kids can type in any character they imagine—a flying cat, a brave robot, or a ninja princess.
* **Flexible Storytelling:** The AI adapts the plot based on the custom characters and settings provided by the child.
* **Child-Safe AI:** Powered by strictly moderated AI prompts to ensure 100% kid-friendly and educational content.
* **Real-time Magic:** Watch the story magically type itself out on the screen, word by word.

## 🛠️ Tech Stack

* **Framework:** Ruby on Rails
* **Database:** sqlite3
* **Real-time Updates:** Hotwire (Turbo Streams & Stimulus JS)
* **Background Processing:** Sidekiq & Redis
* **AI Engine:** OpenAI / LLM integration via `langchainrb`
